# hush. — Landing page

> L'IA qui répond à vos DMs Instagram comme vous le feriez.

Landing page officielle du projet **hush**, un agent IA conversationnel dédié à la gestion des DMs Instagram des marques DTC (Direct-To-Consumer).

## 📂 Contenu

- `index.html` — Page unique self-contained (HTML + CSS + JS dans un seul fichier)

## 🚀 Déploiement

Cette landing est hébergée via **GitHub Pages** à l'adresse :
👉 https://[username].github.io/[nom-du-repo]/

Quand le domaine `hush.io` sera acquis, le site sera accessible sur :
👉 https://hush.io

## 🛠️ Stack technique

- **Single HTML file** (zero dépendance autre que Google Fonts)
- **Inter Variable Font** via Google Fonts
- **Vanilla JS** pour animations Intersection Observer
- **CSS moderne** : custom properties, grid, flexbox, animations
- **Mobile-first responsive**
- **Performance** : ~50KB, charge < 1s

## 🎨 Design

Inspiré des références B2B SaaS modernes :
- **Linear** (minimalisme dramatique)
- **Anthropic** (apaisement, intellectualité)
- **Stripe** (gradients sophistiqués)
- **Vercel** (typographie audacieuse)

Palette :
- Background : `#0A0A0A` (noir profond)
- Texte : `#F5F1EB` (off-white chaleureux)
- Accent : `#A88FAC` (mauve doux)
